## Статусы решений

- **Подтверждено** — можно проектировать и разрабатывать без дополнительного
  согласования.
- **Рабочая гипотеза** — можно прототипировать, но нельзя считать окончательным
  правилом.
- **Открытый вопрос** — требуется дизайнерское решение до реализации.
- **Вне текущего этапа** — идея сохранена, но сейчас не прорабатывается.

При изменении подтверждённого решения нужно обновить связанный документ и
сообщить другому участнику, если изменение затрагивает общий контракт.
